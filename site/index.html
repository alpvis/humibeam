Screenshots in Claude Code
über SSH einfügen.

humibeam ist der SSH-Client, der für agentische CLIs gebaut ist: Bilder, Sprache und Dateien landen in einer Remote-claude-Session — über eine Verbindung, ohne Server-Installation.

humibeam laden →
macOS 14+ · Apple Silicon & Intel · eigener OpenAI-Key für die KI-Funktionen

✨ Das, was kein SSH-Client kann

Drück Cmd + V mit einem Screenshot im Terminal — humibeam lädt ihn über die bestehende SSH-Verbindung hoch und tippt den Pfad in die laufende claude-Session. Claude Code liest das Bild. Funktioniert auf jedem nackten Ubuntu-Server, ohne dort etwas zu installieren.

Terminal
Tabs, Split-Panes, Suche, Themes, Reconnect, Broadcast — auf iTerm2-Niveau.
Dateien
SCP/SFTP-Browser: Up/Download, mkdir, chmod, Umbenennen, Ordner als .tar.gz, Remote-Editor.
KI & Sprache
Diktat ins Terminal, Ausgabe erklären, Fehler beheben, Befehle vorschlagen.
📸

Screenshot-Paste

Bild kopieren → Cmd+V → Upload → Pfad in der Claude-Session.

🎙️

Voice-Diktat

Hotkey halten, sprechen — der Text wird direkt ins Terminal getippt.

🗂️

Datei-Browser

Drag & Drop rein/raus, Remote-Dateien direkt bearbeiten, Lesezeichen.

🔀

Port-Forwarding

ssh -L mit einem Klick — lokale Ports sicher durch den Tunnel.

🧠

KI-Hilfe

Terminal-Ausgabe erklären, Fehler deuten, passende Befehle vorschlagen.

🔑

Sicher

Schlüssel im macOS-Keychain, eigener humibeam-Key oder bestehende SSH-Keys.
